# 繁體中文版本

## 🎉 項目介紹

**ProcTrace** 是一款專注於系統安全分析的行程追蹤工具，幫助使用者快速識別系統中的可疑行程和潛在威脅。

## ✨ 核心特性

- 🔍 行程掃描 - 全系統行程資訊獲取與分析
- 🎯 威脅評分 - 0-100分智能威脅評估系統
- ⚠️ 風險分級 - 高/中/低/安全 四級風險分類
- 🌐 Web視覺化 - 現代化儀表板即時展示

## 🚀 快速開始

```bash
# 安裝
pip install proctrace

# 掃描所有行程
proctrace scan

# 啟動Web介面
proctrace web
```
